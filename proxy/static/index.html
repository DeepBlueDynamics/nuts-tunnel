how it works

1

sign up

enter your email. we send a magic link. no passwords, no OAuth, no friction.

2

create a token

generate an API token in your dashboard. claim a unique tunnel name. done.

3

connect

run the client. your local service is instantly live at tunnel.nuts.services/yourname/

path-based routing

no wildcard DNS or custom domains needed. every tunnel gets a clean path under one domain.

auto-reconnect

client reconnects with exponential backoff. tunnels survive network blips and laptop sleeps.

self-hostable

deploy your own proxy to any Cloud Run project. one Dockerfile, one env var, done.

multi-service

one client, many services. expose your entire dev stack through a single tunnel connection.
